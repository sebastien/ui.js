What makes UI.js stand out?

			
    
				
  • 
					Ready to use, all you need is to write
					an HTML file, you can define and import components, interact
					with APIs and write complex interactions all within a single file.
				
    • 
				
  • 
					No transpiler or build step required,
					UI.js is directly usable from your browser by importing
					its main JavaScript file and optional CSS.
				
    • 
				
  • 
					Batteries included: rendering,
					reactivity, state management, CSS utilities, standard
					components and functional primitives are all there by
					default.
				
    • 
				
  • 
					Incremental, granular rendering,
					Updates to the UI are driven by data changes which translated into
					UI changes for great interactive speed.
