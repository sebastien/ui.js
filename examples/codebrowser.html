‚óè SelectorScope

A selector input represents one specific selection in a selector, which is a collection of inputs. Selector inputs can select from the local state (@ prefixed, like @status), or from the global state either in an absolute way (no prefix like application.name) or relative (. prefix like .label). 

    export const SelectorScope = Object.freeze({
      Local: "@",
      Relative: ".",
      Absolute: "/",
      Key: "#",
    });
